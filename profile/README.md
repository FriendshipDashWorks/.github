# FriendshipDashWorks

Minecraft 服务端与客户端生态工具组织 —— 专注 Paper 插件、Fabric 模组与多人体验增强。

---

## 关于我们

**FriendshipDashWorks** 致力于为 Minecraft 服务器与客户端提供高质量、实用的模组与插件：

- **协议桥接**：在服务端补齐客户端 mod 的网络协议，降低玩家装包成本
- **多人体验**：远景 LOD、快捷容器、JEI 配方同步等
- **玩法插件**：回合制竞技等服务端玩法
